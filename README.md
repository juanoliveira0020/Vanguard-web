Vanguard Parking Web

Este projeto é uma aplicação web para o gerenciamento de um sistema de estacionamento. Ele oferece funcionalidades para o cadastro de veículos, controle de entrada e saída, visualização de veículos ativos no pátio, além de exibir o histórico de veículos e informações sobre o faturamento.

Funcionalidades

Cadastro de Veículo: Permite registrar a entrada de veículos no estacionamento.

Controle de Entrada/Saída: Registra a entrada e saída de veículos, com cálculos de tempo e valor.

Veículos Ativos: Exibe a lista de veículos no pátio com informações como placa, entrada e saída.

Histórico de Veículos: Exibe um histórico de veículos que passaram pelo estacionamento, com filtro por data.

Faturamento: Exibe o total e média do faturamento, com a opção de filtrar por intervalo de datas.

Tecnologias

React: Framework utilizado para a construção da interface.

React Router DOM: Para navegação entre as páginas.

HTML5 & CSS3: Estrutura e estilo da aplicação.

Hooks: Utilização de hooks como useState, useEffect e useNavigate para gerenciar estado e navegação.

Como rodar o projeto

Clone o repositório:

git clone https://github.com/seu-usuario/vanguard-parking.git


Instale as dependências:

npm install


Inicie o servidor de desenvolvimento:

npm start
